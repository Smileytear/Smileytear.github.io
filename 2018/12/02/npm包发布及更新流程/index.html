npm包发布及更新流程

一、安装nodejs

这一步不多说

二、创建一个npm包

1.创建一个npm项目(给包起名字前先去npm网站搜索下，看看名字有没有被占用)

2.编写发布的内容，在根目录下创建lib文件夹和index.js，进入lib文件夹新建hello.js和index.js，代码分别如下

3.添加test模块,在根目录下新建文件夹test

4.在根目录下运行npm run test,输出’hello world!’则代码运行正确

此时文件目录如下所示

三、发布一个npm包

1.申请一个npm账号，不多说
2.进入项目根目录，登录npm账号

3.发布npm包 npm publish
如果有如下错误，是因为新申请的npm账号没有核实邮箱，打开你的注册邮箱，点开npm官方发给你的核实邮件，核实一下邮箱地址就好了
)
如果发布成功，则如下所示

进入npm个人中心，就可以看到自己发布的包了

四、使用npm包

npm i <package-name>

可以自己建一个项目，测试一下自己刚刚发布的包

五、更新npm包

刚刚发布的包没有Readme.md,现在通过更新来添加
在根目录下新建Readme.md,内容参考大神发布的包的写法
更新npm包也是使用npm publish，要记得更改package.json里的版本号，不然会报错,更新成功后如下所示
