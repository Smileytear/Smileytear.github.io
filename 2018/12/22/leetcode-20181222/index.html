Unique Morse Code Words

Posted on 2018-12-22 | In leetcode study everyday

International Morse Code defines a standard encoding where each letter is mapped to a series of dots and dashes, ad follows: “a” maps to “.-“, “b” maps to “-…”, “c” maps to “-.-.”, and so on.

For convenience, the full table for the 26 letters of the English alphabet is given below:

Now, given a list of words, each word can be written as a concatenation of the Morse code of each letter. For example, “cba” can be written as “-.-..–…”, (which is the concatenation “-.-.” + “-…” + “.-“). We’ll call such a concatenation, the transformation of a word.

Return the number of different transformation among all words we have.

Note:

  • The length of words will be at most 100.
  • Each words[i] will have length in range [1,12].
  • words[i] will only consist of lowercase letters.

Solution 2 方法中ES6语法简单说明:

  1. reduce()

    接受一个函数作为累加器，数组中的每个值从左到右开始缩减，最终计算为一个值

    array.reduce(function(total, currentValue, currentIndex, arr), initialValue)

    total: 必须，初始值

    currentValue: 必须，当前值

    initialValue: 可选，传递给函数的初始值

  2. Set 数据结构

    ES6 提供了新的数据结构Set。它类似于数组，但是成员的值都是唯一的，没有重复的值。

    Set本身是一个构造函数(并不是数组)，用来生成Set数据结构。

    Set 结构的实例有以下属性：

    • Set.prototype.constructor: 构造函数，默认就是Set函数。
    • Set.prototype.size: 返回Set实例的成员总数

    Set 实例的方法(部分)：

    • add(value): 添加某个值，返回Set结构本身。

    • delete(value): 删除某个值，返回一个布尔值，表示删除是否成功。

    • has(value): 返回一个布尔值，表示该值是否为Set的成员。

    • clear(): 清除所有成员，没有返回值。

    • keys(): 返回键名的遍历器

    • values(): 返回键值的遍历器

    • entries(): 返回键值对的遍历器

      Set的遍历顺序就是插入顺序，比如，使用Set保存一个回调函数表，调用时就能保证按照添加顺序调用

      keys方法、values方法、entries方法返回的都是遍历器对象，由于Set结构没有键名，只有键值，所以Keys方法和values方法的行为完全一致, entries方法返回的遍历器，同时包括键名和键值,所以每次输出一个数组，它的两个成员完全相等。

    • forEach(): 使用回调函数遍历每个成员
